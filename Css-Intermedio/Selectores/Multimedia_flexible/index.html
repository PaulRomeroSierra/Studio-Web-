Ahora para hacer que un video sea responsivo existen dos formas

Primer forma

.container{
position: relative;
max-width: 100%;
padding-top: 56.25%;
height: 0;}

video{
position: absolute;
top:0;
left: 0;
max-width: 100%;}

Segunda forma

.container{
max-width:100%;
aspect-ratio:16/9;}

video{
max-width:100%;}
