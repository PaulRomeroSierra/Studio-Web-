Features queries

¿Cómo funcionan?

Las feature queries usan la regla @supports. Dentro de esta regla, defines una condición que el navegador evaluará. Si la característica está disponible, los estilos dentro de la consulta se aplican; si no, se ignoran.

@supports (propiedad: valor) { /* Estilos que se aplican si la propiedad es compatible */ }

Operadores en feature queries

Negación (not): Aplica estilos si una característica no es compatible. @supports not (display: grid) { .aside { display: block; } }

Múltiples condiciones (and, or):

Puedes combinar condiciones para hacer pruebas más complejas. lo @supports (display: flex) and (display: grid) { .aside { background-color: blue; } } @supports (display: grid) or (display: flex) { .aside { background-color: yellow; } }
