Transiciones

Transition-property: Especifica qué propiedades cambiarán gradualmente. Ejemplo: color, background-color, width, etc. También puedes usar all para afectar todas las propiedades animabless. Pasa el cursor por encima para ver el efecto

Transition-duration: Define cuánto tiempo durará la transición. Se especifica en segundos (s) o milisegundos (ms). Ejemplo: 0.5s.

transition-property: color; transition-duration: 0.5s; transition-property: background; transition-duration: 0.5s;

box

Transition-delay: Establece cuánto tiempo esperar antes de comenzar la transición. Ejemplo: 1s.

transition-delay: 1s;

box

Transition-timing-function: Determina cómo será el ritmo de la transición. Los valores comunes son:

  • Ease (por defecto): Inicia lenta, acelera y luego desacelera.
  • linear: Velocidad constante.
  • Ease-in: Comienza lenta.
  • Ease-out: Termina lenta
  • Ease-in-out: Comienza y termina lenta.
  • También puedes usar funciones personalizadas como cubic-bezier. Cubic-bezier
Saca el cursor de la pantalla para ver el efecto
transition-timing-function: ease-out;
transition-timing-function: cubic-bezier(.17,.67,.96,.15);
