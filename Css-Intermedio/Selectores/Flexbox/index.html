Flexbox

¡Claro! Flexbox es un modelo de diseño en CSS que facilita la alineación, distribución y organización de elementos dentro de un contenedor. Su nombre completo es Flexible Box Layout y está diseñado para manejar diseños en una sola dimensión: ya sea en filas (horizontal) o en columnas (vertical).

¿Cómo funciona?

Para usar Flexbox, debes aplicar la propiedad display: flex a un contenedor. Esto convierte a los hijos directos de ese contenedor en elementos flexibles.

.container { display: flex; }

Propiedades principales

Flexbox tiene propiedades para el contenedor flex y para los elementos flexibles (hijos).

Flex-direction

Define la dirección principal en la que se colocarán los elementos. Valores:

  • row (por defecto): Horizontal, de izquierda a derecha.
  • row-reverse: Horizontal, de derecha a izquierda.
  • column: Vertical, de arriba hacia abajo.
  • column-reverse: Vertical, de abajo hacia arriba.

Justify-content

Controla cómo se distribuyen los elementos en la línea principal. Valores:

  • flex-start (por defecto): Todos los elementos al inicio.
  • flex-end: Todos los elementos al final.
  • center: Elementos centrados.
  • space-between: Espacio igual entre los elementos.
  • space-around: Espacios iguales alrededor de cada elemento.
  • space-envenlly: Espacios iguales.

Align-items

Alinea los elementos en la línea transversal (perpendicular). Valores:

  • stretch (por defecto): Los elementos se estiran para llenar el contenedor.
  • flex-start: Alineados al inicio.
  • flex-end: Alineados al final.
  • center: Alineados en el centro.
  • baseline: Basado en la línea base del texto.

Aling-content

Solo se aplica si hay múltiples líneas. Define cómo se distribuyen las líneas en el eje transversal. Valores: Igual que los de align-items.

Flex-wrap

Controla si los elementos se envuelven al desbordar el contenedor. Valores:

  • nowrap (por defecto): No se envuelven.
  • wrap: Los elementos se envuelven al siguiente renglón o columna.
  • wrap-reverse: Igual que wrap, pero en dirección inversa.
