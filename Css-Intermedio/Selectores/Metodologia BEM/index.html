Metodologia BEM

Block element modificier

Block

  • Representa un componente independiente y reutilizable.
  • Ejemplo: menu, button, form.

Elemento

  • Es una parte del bloque que no puede existir fuera de él.
  • Se conecta al bloque con dos guiones bajos (__).
  • Ejemplo: menu__item, button__icon.

Modifier

  • Representa una variación o estado del bloque o del elemento.
  • Se conecta al bloque o elemento con dos guiones (--).
  • menu--vertical, button--disabled.
