Definir el contenedor "observable" Para usar Container Queries, primero debes indicar que un contenedor es "observable". Esto se hace con la propiedad container-type. Por ejemplo:

.container { container-type: inline-size; }

  • Inline-size: Observa el ancho del contenedor.
  • Size: Observa tanto el ancho como el alto.

Escribir las reglas de consulta

Usas @container para definir las condiciones bajo las cuales aplicar los estilos, basado en las dimensiones del contenedor. Ejemplo:

@container (min-width: 400px) { .child { font-size: 2rem; background-color: lightblue; } }
