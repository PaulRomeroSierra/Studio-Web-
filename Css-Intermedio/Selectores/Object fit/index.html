Object fit

Las propiedades object-fit y object-position son ideales para controlar cómo se muestran imágenes o videos dentro de un contenedor en CSS.

object-fit Controla cómo se ajusta el contenido dentro de su contenedor. Los valores más comunes son:

fill: El contenido se estira para llenar el contenedor. Es el valor por defecto.

contain: Escala el contenido para que quepa completamente dentro del contenedor, manteniendo sus proporciones.

cover: Escala el contenido para llenar completamente el contenedor, pero puede recortarlo para mantener las proporciones.

none: El contenido no se escala. Mantiene su tamaño original.

scale-down: Es como none o contain, dependiendo de cuál necesite menos escalado.
