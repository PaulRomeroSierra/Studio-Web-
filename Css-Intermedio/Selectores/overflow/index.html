Overflow

La propiedad overflow en CSS se utiliza para controlar qué ocurre cuando el contenido de un elemento supera las dimensiones de su contenedor.

Visible (por defecto): El contenido que desborda no se recorta y será visible fuera del contenedor. Ejemplo: Ideal cuando no quieres limitar el contenido visible.
