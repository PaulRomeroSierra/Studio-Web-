El atributo srcset en HTML se utiliza en elementos de imagen () para especificar un conjunto de imágenes alternativas que el navegador puede usar en función de las características del dispositivo o del entorno (como la resolución de pantalla o el ancho de la ventana).

img src="dalto.png" srcset="dalto-small.png 400w, dalto-medium.png 600w, dalto.png 1000w" sizes="(max-width: 400) 300px, (max-width: 600px) 500px,900px"

Es especialmente útil para implementar imágenes responsivas, donde diferentes tamaños de imágenes se cargan según las necesidades del usuario, optimizando así el rendimiento de tu sitio web.

Estructura básica de srcset Desglose de atributos: src:

Especifica la imagen predeterminada que se cargará si el navegador no soporta srcset. srcset:

Define una lista de imágenes y el tamaño correspondiente que el navegador puede elegir.

Cada entrada tiene: URL de la imagen (por ejemplo, imagen-grande.jpg). Descriptor: w (ancho en píxeles): Indica el ancho de la imagen, como 1200w. x (densidad de píxeles): Indica la densidad de la imagen, como 2x. sizes:

Especifica cómo debe usarse el espacio disponible (normalmente en relación al ancho de la ventana). Por ejemplo, (max-width: 600px) 400px significa que, si el ancho de la ventana es menor a 600px, la imagen será renderizada con un ancho de 400px.
