Etiqueta picture

Atributo media y srcset

La etiqueta picture se utiliza para proporcionar varias versiones de una imagen con diferentes condiciones, como tamaños de pantalla o resoluciones, y el navegador elige automáticamente la más adecuada. Esto es útil para diseño adaptable y optimización de recursos.

El elemento source se utiliza dentro de picture para definir las diferentes fuentes de la imagen con atributos como srcset y media.

Cómo funciona:

Si el ancho de la ventana es de al menos 400px, se usará imagen-pequeña.jpg.

Si el ancho de la ventana es menor a 600px, se usará imagen-mediana.jpg.

Si el navegador no soporta picture, se mostrará la imagen predeterminada en img.
