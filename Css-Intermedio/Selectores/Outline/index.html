Outline

¿Qué es?

outline crea un borde alrededor del elemento, pero no afecta su tamaño ni desplaza a otros elementos, ya que no forma parte del modelo de caja.

    Propiedades asociadas:

  • outline-width: Define el grosor del contorno.
  • outline-style: Define el estilo del contorno (solid, dotted, dashed, etc.).
  • outline-color: Define el color del contorno.
  • outline-offset: Agrega espacio entre el contorno y el borde del elemento.
  • outline: Propiedad abreviada que combina outline-width, outline-style y outline-color, si queremos sacar el contorno de un input lo hacemos con oulie:none; pero tenemos que darle la pseudoclase focus-visible para no tener problemas de accesibilidad.
